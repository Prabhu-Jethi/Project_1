# Data Privacy in IoT

## Objectives

- Protect data generated by IoT sensors and devices.

- Implement encryption techniques to ensure confidentiality and integrity.

- Detect unauthorized access or tampering attempts.

- Secure data during storage, transmission, and processing.

## Technologies used

- Python

- Pandas/Numpy - Data Processing

- Matplotlib/Seaborn - Plotting/Visualization

- Cryptography/Pycrypto - Data Encryption

- Faker - Generate Iot dataset

- Keras/Tensorflow - Model creation


# How to run the project

### 1. Clone the repository
 ### git clone https://github.com/your-username/your-repo-name.git

### 2. Navigate to project folder
 ### cd DATAPRIVACY_IN_IOT

### 3. Install required libraries
 ### pip install -r requirements.txt

### 4. Run notebook
 ### jupyter notebook main.ipynb
